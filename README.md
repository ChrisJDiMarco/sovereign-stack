## What's in the stack

- **[Hermes](https://github.com/NousResearch/hermes-agent)** — Persistent memory agent. SQLite across every session, FTS5 search, 160+ skills, 8 LLM providers.
- **[Paperclip](https://github.com/paperclipai/paperclip)** — Open-source AI company OS. Visual org chart, heartbeat scheduler, per-agent cost controls.
- **[Hetzner CX22](https://hetzner.com)** — $5/month VPS. Ubuntu 24.04, PM2, Caddy for HTTPS.
- **Hermes Gateway** — Telegram listener. DM your CEO agent from your phone in under 1 second.
- **[VibeVoice](https://github.com/microsoft/VibeVoice)** — Microsoft's open-source ASR. Speak to your agents on Mac via the Vibing app.
