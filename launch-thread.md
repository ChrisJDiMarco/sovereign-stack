**Tweet 2 — The problem**

Most "AI agent" setups forget everything when the session ends. You're back to zero every time.

The fix: Hermes — Nous Research's open-source agent with persistent SQLite memory and FTS5 recall across every session. 60+ built-in tools out of the box.

**Tweet 3 — The stack**

The Sovereign Stack runs 3 agents 24/7 on a ~$5/month VPS:

→ CEO: orchestrates, remembers, decides
→ Research: live web intel on demand
→ Builder: Claude Code + a task queue

All governed by Paperclip — visual org chart, per-agent cost caps, approval flows. Also open source.

**Tweet 5 — Rent vs. own**

I wanted the opposite: memory, data, and the cost ceiling all on hardware I control. Your VPS, your SQLite, your API keys with hard caps. Nothing leaves the box.

That's the whole pitch.
